/************   2.2接口    ************/

1. 订单列表页接口: home/order/index
    请求方式: GET
    请求参数: sid     门店id
        status  订单状态
        page    页数 初始值0

2. 订单详情接口: home/order/detail/id
    请求方式: GET
    请求参数: id   订单id
3. 更改订单状态 home/order/doapply/id/1/status/1
    请求方式: get
    请求参数: id 订单id  status 订单状态

/************   2.1接口    ************/

1. 员工列表: home/staff/index/sid/1/style/1
    请求方式: get
    请求参数: sid   门店id
        style 1员工  2司机
        search 搜索 员工姓名的时候带

2. 添加员工: home/staff/add
    请求方式: POST
    请求参数: sid    门店id
        style     1  1员工   2 司机
        name:  员工姓名
        mobile 手机号
        password  密码
        repassword 确认密码
        iscall     是否打电话

3. 修改员工: home/staff/edit
    请求方式: POST
    请求参数: id    员工id
        sid    门店id
        style     1  1员工   2 司机
        name:  员工姓名
        mobile 手机号
        password  密码
        repassword 确认密码
        iscall     是否打电话

/************    1.31接口    ************/

1. 门店登陆: home/login/index
    请求方式: post
    请求参数: mobile: 手机号
        password: 密码
        数据格式:   style: 1 公司 2 个人
        {
            "code": 1,
            "msg": "登陆成功!",
            "data": {
                "id": 1,
                "name": "门店一",
                "style": "2"
            }
        }

2. 门店注册(申请): home/login/register
    注册短信接口:  home/login/sendSMS
    请求方式: post
    请求参数:
        mobile:  手机号
        password: 密码
        code:   	验证码
        name: 	门店名字
        style:   类型 1 公司 2个人
        idcard: 	身份证
        license: 营业执照或者驾驶证
        address: 地址

3. 找回密码:    home/login/forget
    找回短信接口: home/login/sendPubSMS
    请求方式: post
    请求参数: mobile:  手机号
        password: 密码
        repassword 2次密码

4. 门店首页列表  home/index/index_list
    请求方式: GET;

5. 门店上下班  home/index/index/id/1/open/1
    请求方式: GET
    请求参数: id 门店id
        open 1 上班  2 下班
